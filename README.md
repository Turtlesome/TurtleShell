# TurtleShell
My learning project for creating a shell in C

Arguments and options are seperated into a hashmap.
Adding commands made with trees and self-balancing algorithm.

Available commands are:
1.  pwd
2.  cd
3.  mkdir
4.  rmdir
5.  mkfile
6.  rmfile
7.  ls with -a option
8.  cp
9.  mv
10.  exit
